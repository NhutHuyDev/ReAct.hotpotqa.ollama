## Summary

- Clone ReAct (Synergizing Reasoning and Acting in Language Models) with Ollama.

## Prerequisites
- Install **Python** environment
- Install **gym**
```
    pip install gym
```
- Install **bs4**
```
    pip install bs4
```
- Install **ollama** for Python
```
    pip install ollama
```
- Install **Ollama** and pull model from it 
```
    ollama pull phi3
```
